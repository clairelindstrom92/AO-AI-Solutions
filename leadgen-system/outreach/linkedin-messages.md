## GENERAL RULES

1. Never pitch in a connection request
2. Never send more than 3 messages in a sequence without a reply
3. Always personalize the first line — generic = deleted
4. Goal of every message: earn the NEXT step, not a sale
5. If someone says "not interested" — thank them and close professionally. Never argue.
6. Keep messages under 150 words when possible (mobile-first)
7. Voice notes and Loom videos get 3x higher response rates — use them for warm prospects
